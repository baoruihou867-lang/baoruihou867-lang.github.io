# 工程英语工具站

这是一个免费起步的静态网站，当前适合部署到 GitHub Pages。

## 本地预览

直接在浏览器打开 `index.html` 即可预览。也可以运行一个本地服务器：

```powershell
python -m http.server 8000
```

然后访问 `http://localhost:8000`。
