# 工程英语学习站

这是一个免费起步的静态网站模板，适合部署到 GitHub Pages、Cloudflare Pages、Netlify 或 Vercel。

## 本地预览

直接在浏览器打开 `index.html` 即可预览。也可以运行一个本地服务器：

```powershell
python -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 上线前要改的地方

1. 把 `index.html` 里的 `https://example.com/` 改成你的正式网址。
2. 把 `robots.txt` 和 `sitemap.xml` 里的 `https://example.com` 改成你的正式网址。
3. 继续增加工程英语词条、文章和行业专题页面。

## 免费上线建议

优先使用 Cloudflare Pages 或 GitHub Pages。前期可以用平台提供的免费二级域名，等网站内容稳定后再购买自己的域名。
